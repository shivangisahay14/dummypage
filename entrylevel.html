The Hogwarts Mystery

INSTRUCTIONS

1. The Triwizard Tournament will be taking place at Hogwarts this year.
2. It will have 3 stages:

a. Getting golden egg from the dragon.
b. Saving one person from the Merpeople at the bottom of the lake.
c. The Triwizard maze.

3. But before entering the main tournament you will have to pass a basic round.
4. Before each coding question, you will encounter a harry potter related question. If answered correctly, you would get a hint to the coding problem. However, even if answered wrong, you would still be redirected to the coding question.
5. NOTE : It is advisable to play via laptop.
